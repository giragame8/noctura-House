🏚️ Noctura House
Jeu narratif sombre – Développement solo
🎯 Présentation
Noctura House est un projet de jeu narratif sombre et mystérieux, développé entièrement en solo sur mon temps libre.
Le développement avance progressivement, au rythme de mes disponibilités et de mes envies créatives.

L’objectif final est de créer une expérience immersive, psychologique et oppressante, centrée sur l’exploration d’une maison aux secrets enfouis.

🧪 État actuel du projet : Démo technique
Pour l’instant, le jeu se présente sous la forme d’une démo jouable en ligne de commande (CMD).

Cette démo inclut :
🌀 Un labyrinthe généré aléatoirement à chaque lancement

🧭 Une navigation simple au clavier

🌑 Une ambiance minimaliste mais volontairement oppressante

🧱 Un système de déplacement et de détection des murs

Cette version sert de prototype pour tester les mécaniques de déplacement, la génération procédurale et la structure globale du gameplay.

🛠️ Technologies utilisées
C#

.NET Framework / Visual Studio 2019

Console / CMD

Génération procédurale simple

🚧 Développement en cours
Le projet avance petit à petit.
Les prochaines étapes prévues :


👁️ Apparition d’événements aléatoires (sons, illusions, entités)
